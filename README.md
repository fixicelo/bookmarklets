# Bookmarklets

### Toggle metadata visibility of YouTube Shorts
```javascript
document.querySelectorAll('#experiment-overlay div.metadata-container').forEach(e => {
  e.style.visibility = (e.style.visibility === 'hidden') ? 'visible' : 'hidden';
});
```
